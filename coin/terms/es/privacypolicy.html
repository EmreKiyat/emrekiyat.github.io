Política de Privacidad (Privacy Policy)

La privacidad de nuestros usuarios es importante para nosotros. Esta página de política de privacidad describe los tipos de información personal que recibe y recopila CandleStick Patterns for Coins App y cómo se utiliza.

Recogida y Uso de la Información

La aplicación CandleStick Patterns for Coins puede utilizar herramientas o software de análisis de terceros para recopilar y utilizar automáticamente datos no personales que no le identifican directamente. Los datos no personales pueden ser los siguientes o más: la dirección de protocolo de Internet (IP) de su ordenador, el tipo de navegador, la versión del navegador, las páginas de nuestra aplicación que visita, la hora y la fecha de su visita, el tiempo que pasa en esas páginas y otras estadísticas, la plataforma de software del dispositivo y el firmware, el identificador único del dispositivo u otro identificador del dispositivo. La aplicación CandleStick Patterns for Coins puede utilizar los datos no personales recopilados para analizar el uso de los Servicios y mejorarlos.

Su lista de notificaciones y Watch se almacenan en los servicios de Google podría entregar notificaciones de forma fiable y con una latencia mínima. También se almacena el token APN del dispositivo Apple o el token Firebase Cloud Messaging (FCM) de Google. El token APN o FCM es una larga cadena de caracteres para enviar notificaciones a tu dispositivo. El token es completamente anónimo y no revela nada sobre tu dispositivo o identidad.

Terceros proveedores de servicios utilizados por la aplicación;

  • Google Analytics for Firebase
  • Firebase Crashlytics
  • RevenueCat Subscription
  • Google Firestore Database
